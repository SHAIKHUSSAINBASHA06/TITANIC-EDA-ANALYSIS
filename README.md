# 🚢 Titanic EDA Analysis

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to identify factors that influenced passenger survival. The analysis includes data cleaning, missing value treatment, outlier detection, feature analysis, and visualizations.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Most passengers did not survive the disaster.
- Female passengers had a higher survival rate than males.
- Children survived more frequently than adults.
- First-class passengers had better survival chances.
- Higher fare-paying passengers were more likely to survive.

## Project Workflow
- Data Exploration
- Data Cleaning
- Missing Value Handling
- Outlier Analysis (IQR Method)
- Survival Analysis
- Data Visualization

This project demonstrates how EDA can uncover meaningful patterns and insights from real-world datasets.
